<h2><a href="https://www.geeksforgeeks.org/problems/shortest-path-with-at-most-k-nodes/1">Shortest Path with at most k Nodes</a></h2><h3>Difficulty Level : Difficulty: Medium</h3><hr><div class="problems_problem_content__Xm_eO" style="--text-color: var(--problem-text-color);"><p><span style="font-size: 18px;">Given a weighted directed graph of n vertices represented by a 2D array <strong>edges[][]</strong>, where each element edges[i] = {u, v, w} denotes a directed edge from vertex <strong>u</strong> to vertex <strong>v</strong> with cost <strong>w</strong>, and three integers <strong>src</strong>, <strong>dst</strong>, and <strong>k</strong>.<br></span><span style="font-size: 18px;">Find the minimum cost to travel from src to dst using a path containing at most k nodes. If no such path exists, return -1.</span></p>
<p><span style="font-size: 14pt;"><strong>Note</strong>: The cost from city A to B may be different from the price from city B to A.</span></p>
<p><strong><span style="font-size: 14pt;">Example:&nbsp;</span></strong></p>
<pre><span style="font-size: 18px;"><strong>Input: </strong>n = 6, edges[][] = [[0, 1, 10], [1, 2, 20], [1, 3, 10], [2, 5, 30], [3, 4, 10], [4, 5, 10]], src = 0, dst = 5, k = 2
<strong>Output: </strong>60
<strong>Explanation: </strong></span><span style="font-size: 18px;">There can be a route marked with a red arrow that takes cost =  10+10+10+10 = 40 using three nodes. And route marked with green arrow takes cost = 10+20+30=60 using two nodes. Since there can be at most 2 nodes, the answer will be 60.<br></span>            <img src="https://media.geeksforgeeks.org/img-practice/prod/addEditProblem/926932/Web/Other/blobid0_1778302811.png" width="335" height="183"> </pre>
<pre><span style="font-size: 18px;"><strong>Input: </strong>n = 3, edges[][] = [[0, 1, 10], [0, 2, 50], [1, 2, 10]], src = 0, dst = 2, k = 1
<strong>Output: </strong>20<br><strong>Explanation:</strong> Since k is 1, the the green-colored path can be taken with a minimum cost of 20.<br>                   <img src="https://media.geeksforgeeks.org/img-practice/prod/addEditProblem/926932/Web/Other/blobid0_1778478563.png" width="202" height="154"><br></span></pre>
